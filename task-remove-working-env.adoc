= NetApp ConsoleからGoogle Cloud NetApp Volumes を削除する
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Google Cloud NetApp Volumesサブスクリプションと既存のすべてのボリュームをNetApp Consoleから削除できます。ボリュームは削除されず、 NetApp Console内のすべてのプロジェクトから削除され、後で再検出できるようになります。


TIP: NetApp ConsoleからGoogle Cloud NetApp Volumesを削除することはできません。このアクションは、Google Cloud Console を使用して実行します。

.手順
. システムを開きます。
. クリックimage:screenshot_gallery_options.gif["3 つのドットがあるアイコン。"]ページ上部のボタンをクリックし、[削除] をクリックします。
